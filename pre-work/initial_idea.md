 # Origin of and Initial Ideas for Solicited Advice Agent
 
 I have a WhatsApp group with some friends of mine. We're all in our early 50s. I am an AI professional, implementing projects, building with AI. One of the other gentlemen in our group is also a heavy user of AI with quad code and codex and so forth. But many of the other members of the group are AI curious. In fact, the name of the group is AI curious. The purpose of that WhatsApp group is for a group of friends to ask questions about AI and how to use it, what the societal implications are, how they can help their friends, how they can use it for their own lives in a safe place without any judgment. We all trust each other, we've known each other for years and it's a good discussion about all things AI.

I have provided some advice to specific questions from people when they ask about how to get started or how to help a particular friend. I can provide some examples of that, but the feedback that I've gotten on my responses has been very strong. And say wow, this is incredible advice, this is gold, you should write a book. So that's awesome.

My first thought was not to write a book but maybe to write a blog and have the WhatsApp group help me write that blog. Then I'm thinking, well, what if I could create an agent that reads that WhatsApp group and finds the questions and my answers and then formulates those answers, cleans them up, turns them into something publicly consumable, and then packages it up into a LinkedIn post, a Twitter post, a blog post, whatever social media consumption or social media posting we want to do and publish it. So that the world could get the benefit of the conversation that we're having inside our friend WhatsApp group.

So that was my first idea for an agent. I view this as being very meta in that I'm using an agent to teach people how to use AI based on the questions that they're asking about AI in an AI group. And I'm using that to trigger more questions about AI. So it's a very meta type of thing. I think the idea of building this agent can help me teach people how to build with AI.

But now I have other ideas as well. I would actually like this agent to become a member of the WhatsApp group. I want to teach this agent how I think and how I provide advice to people, and have it monitor the WhatsApp group looking for questions, looking for people that are seeking solicited advice. And that's why the name of this project is Solicited Advice. The name of the agent, so it would join our WhatsApp group and be an active participant, responding to any questions that people ask generally to the group. It could respond to direct questions for sure. It could engage in conversations and go deeper than just the first answer that it gives. But basically, it would be an extension of me, and then we could teach it how to give answers like me, and it could teach me how to give better answers in terms of my advice.

Longer term, I'm thinking that this agent could be a public agent where other people could include it in their group chats about AI. Or we could create a public AI chat where people could congregate and talk about AI. This could be in WhatsApp, it could be in Telegram, it could be in Twitter, it could be on LinkedIn. But it could be a way for people to ask questions about AI. I'm not exactly sure what form it would take, but I think it could have legs beyond just the usage in my personal friend WhatsApp group.

I want to use this as an opportunity for me to learn. This is a personal project, not a business project, at least not yet. I want to learn about how to build interactive chat bots. I want to learn how to build compounding. I want this thing to get better as people use it and receive advice from it, and I want  it to learn from that feedback.

- Encourage them to use AI to teach them how to use AI

- Keep it free of jargon and to make it palatable for a very smart but relatively non-technical audience

So I am just trying to help bring them up this learning curve and remove some of the mystery from this world of AI that is exciting but mysterious. # Ideas for the Solicited Advice Agent

This bot or agent has two main functions:
1. To act as a participant of the WhatsApp group and interact with the members of the group who ask it questions and have conversations with it.
2. It retrieves the best of the conversation and prepares it for generally available content that can be distributed and it distributes it through social media.

Since this is a hobby project, I'd like to do as much as I possibly can with free technology or at least local to my machine before I have to pay some service to move it into production. Because I think that's part of the education and advice that I want to give people is that whenever possible you can experiment with these things for a very low cost or free, and not have to spend a lot of money to learn about AI. Furthermore, I don't want to spend a lot of money because I don't know where this is going to go. I want to do a minimum viable product and see what kind of feedback I get, just like any kind of lean startup approach. So let's try to do this in such a way where I can test it without having to incur any costs in the short term if it can be avoided.

I want to build this with the very latest tools as of April 17, 2026. I am very interested in using things like CLAWD's dispatch and managed agents and the remote control for CLAWD code. These are very interesting to me.

I want to set this up as a real agent with tools that can work in a loop for any query. I am a good coder, I am a good systems thinker, but I have not put together this specific type of system before, so I'll need your help brainstorming it. I want you to walk me through all the questions that I need to think about as I put this together.

The other thing for you to know is that this particular brainstorm document will be part of a public repo that everyone will have access to. It will be another learning tool so that I'm showing my process of how to work with AI to build things that you want. 

I'll be building this on a Windows Dell laptop. I have Cloud Code installed, I have GitHub and a GitHub account, I have all the basics.
